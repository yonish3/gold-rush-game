About this project

This game is using only JavaScript to render and mimic computer moves. The algorithm is based on DFS, which is implemented using a recursive function.

Link to GitHub repository

General structure of generating a matrix
  1. A random matrix is generated with coins and walls
  2. Then DFS algorithm is initiated in a while loop, the loop is braked once the following 2 criteria are fulfilled:
    • Player 1 has a clear path to reach Player 2
    • Players have a clear path to every gold coin in the matrix
General structure of computer moves
  1. An array of gold coins is generated
  2. Then the closest coin to the player (computer) is being calculated using a heuristic method.
  3. The path to that coin is calculated using DFS algorithm, which is initiated step by step using setTimeout async function
  4. If the target coin is taken by the other player, or the other player is blocking the way, all async functions are canceled and a new path is calculated
